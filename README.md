###  /* animate.css */
 专门放动画： class分为三类<br>
 static => 给元素添加 transition: all 1s; <br>
            只要元素样式有发生变动，就会添加渐变效果<br>
 _init => 元素动画初始样式<br>
 _done => 元素动画完成样式<br>

 全局样式
